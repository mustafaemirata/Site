Hakkımda

Hakkımda

Bilgisayar mühendisliği 2. sınıf öğrencisiyim. İlk yıl Java programlama dersi aldım. Gördüğüm Bilgisayar Mühendisliğine Giriş dersiyle Ağ yapıları, ağların birbirleriyle iletişimlerini ve çalışma prensiplerini temel anlamda gördüm. Yaptığımız araştırmalarla şehirler arası ağ iletişim modeli tasarlamıştık. Şu anda Algortima ve Nesnesel Tasarım derslerinde C ve C++ dillerini görüyoruz. Öğrendiğim kurslarda Tensorflow kullanarak Python’da çeşitli veri setlerinde işlemler yaptım ve grafik üzerinde uygulamalar gerçekleştirdim. Java, SQL ve Firebase veritabanı kullanarak çeşitli küçük uygulamalar geliştirdim. Android programlama tarafında Kotlin ile uygulamalar geliştiriyorum. C# .NET ve Entity Framework teknolojilerini kullanarak DevExpress otomosyon sistemlerinde çalışıyorum. Ayrıca son aylarda Linux işletim sisteminde üzerinde kendimi geliştirmeye çalışıyorum.
